Steps:
1) Open TomLosito.xlsx and save Phase A (or Phase B/C) as CSV UTF‑8.
2) In GoFormz: Admin → Developer → Import CSV → Upload file. Map columns if prompted.
3) Run a 5‑row test import. If OK, run the full import.
