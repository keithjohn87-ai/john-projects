# OAuth Setup Instructions

## What You Need
- Any computer with browser (library, friend, work)
- 2 minutes

## Steps

1. **Install rclone** (if not installed)
   - Windows: Download from rclone.org, unzip
   - Mac: `brew install rclone`
   - Linux: `sudo apt install rclone` or download binary

2. **Open terminal/command prompt**
   - Windows: CMD or PowerShell
   - Mac/Linux: Terminal

3. **Run this command:**
   ```
   rclone authorize drive
   ```

4. **Browser opens automatically**
   - Sign in to CharlesCreatorAI@gmail.com
   - Click "Allow" for Drive access
   - Page shows: "Success!"

5. **Copy the code**
   - Looks like: `{"access_token":"...","token_type":"Bearer","refresh_token":"..."}`
   - Select all, copy

6. **Send me the code**
   - Paste in Telegram
   - I'll save it and have full Drive access

## Done
- Takes 2 minutes
- Works forever
- I can then upload/download files automatically
