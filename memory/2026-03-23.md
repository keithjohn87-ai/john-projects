# March 23, 2026 - Hetzner Local Model Setup

## Context
Migrating from OpenAI API to local Ollama on Hetzner CCX23 (16GB RAM) to eliminate API token costs/rate limits.

## Recommended Model
**Qwen2.5-Coder 7B** - Best for templates and Excel work
- Size: ~4.5GB
- RAM used: ~6GB
- Speed: 20-30 tokens/sec
- Download: `ollama pull qwen2.5-coder:7b`

## Setup Commands (Hetzner)

```bash
# Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Start Ollama
ollama serve &

# Pull model
ollama pull qwen2.5-coder:7b

# Test
ollama run qwen2.5-coder:7b
```

## Accuracy Patterns for 7B Models

### 1. Strict System Prompt
```
You are a precise Excel formula generator. 
Rules:
- Output ONLY the formula, no explanation
- Validate all cell references exist before outputting
- Close every parenthesis
- Use absolute references ($A$1) when appropriate
- Double-check: count opening ( and closing ) match
```

### 2. Few-Shot Examples (CRITICAL)
Always provide 2-3 examples before asking for output:

```
Generate an Excel formula to calculate overtime pay.

Example 1:
Input: Hours=45, Rate=$20
Output: =IF(A1>40, (40*B1) + ((A1-40)*B1*1.5), A1*B1)

Example 2:
Input: Hours=38, Rate=$25  
Output: =IF(A1>40, (40*B1) + ((A1-40)*B1*1.5), A1*B1)

Now generate for: Hours={{hours}}, Rate={{rate}}
Output:
```

### 3. Structured Output (JSON Mode)
```
Generate the formula and validation in JSON:
{
  "formula": "...",
  "explanation": "...",
  "validation": {
    "parentheses_balanced": true,
    "cell_references_valid": true
  }
}
```

Parse JSON and validate before using. Retry if validation fails.

### 4. Chain-of-Thought (Break Complex Tasks)
DON'T: "Generate complete invoice template"
DO: Break into steps:
1. "List required columns for invoice"
2. "Generate SUM formulas for totals"
3. "Create header formatting JSON"
Then assemble results.

### 5. Self-Verification Pattern
```
Formula: {generated_formula}

Checklist before outputting:
- [ ] All parentheses closed
- [ ] Cell references match input sheet
- [ ] No syntax errors
- [ ] Test with sample values

If any [ ] is unchecked, fix and recheck.
```

### 6. Create Skill File
Create skill at `/usr/local/lib/node_modules/openclaw/skills/excel-formula-gen/` to automate these patterns.
