GETTING STARTED

1. THIS IS THE INSTRUCTIONS TAB
   - Review these instructions before using the calculator
   - Yellow cells = You fill in
   - Green cells = Auto-calculated
   - Gray cells = Locked/protected

2. HOW TO USE THIS SPREADSHEET:
   a) Go to the "Project Estimator" tab
   b) Enter your project information (yellow cells)
   c) Fill in labor costs, materials, equipment
   d) Add subcontractor estimates
   e) Review the automatic totals and markup calculations
   f) Check your profit margin

3. TRIAL VERSION LIMITS:
   - You can create up to 3 project estimates
   - Print function includes trial watermark
   - Some advanced features (change orders, multi-job reporting) are locked
   - Full version includes: Unlimited projects, advanced reporting, integration with other trackers

4. UNDERSTANDING THE COLORS:
   🟡 YELLOW CELLS: Enter your data here
   🟢 GREEN CELLS: Formulas calculate automatically
   ⚪ WHITE CELLS: Optional notes/descriptions
   ⬜ GRAY CELLS: Locked - do not modify

5. KEY FEATURES:
   ✅ Labor cost calculator with burden
   ✅ Material cost tracking
   ✅ Equipment rental/owned cost allocation
   ✅ Subcontractor cost management
   ✅ Automatic markup calculations
   ✅ Profit margin analysis
   ✅ Per-square-foot costing
   ✅ Bid comparison tools

6. FORMULAS EXPLAINED:
   - Total Cost/Hr = Base Wage × (1 + Burden%)
   - Total Labor Cost = Hours × Cost/Hr
   - Subtotal = Sum of all category costs
   - With Markup = Subtotal × (1 + Markup%)
   - Profit Margin = (Price - Cost) / Price

7. BEST PRACTICES:
   ☑ Always include burden (taxes, insurance, benefits) in labor
   ☑ Add 10-15% contingency for unknowns
   ☑ Track actual vs estimated for future accuracy
   ☑ Update material costs regularly (they change!)
   ☑ Get 3+ subcontractor bids when possible
