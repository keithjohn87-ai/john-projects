# Essential Forms Package

Thank you for purchasing the Essential Forms Package from ContractorPro!

## What's Included

1. **Preliminary_Notice_Template.html**
   - Use this to protect your lien rights
   - Send within 20 days of starting work (varies by state)
   - Required in most states before filing a lien

2. **Mechanics_Lien_Form.html**
   - File this if you haven't been paid
   - Strict deadlines apply (typically 90-120 days)
   - Must be notarized and recorded with county

3. **Release_of_Lien.html**
   - Use when you've received full payment
   - Required to clear property title
   - Record with same county as original lien

4. **Construction_Contract.txt**
   - Basic contract template
   - Customize for your specific jobs
   - Have an attorney review before use

5. **50_State_Lien_Laws_Reference.pdf**
   - Quick reference for lien requirements by state
   - Check your specific state's rules
   - Laws change - verify current requirements

## How to Use These Forms

1. Open HTML files in any web browser
2. Fill in all blank fields
3. Print to PDF or paper
4. Sign where indicated
5. Follow filing instructions on each form

## Important Legal Notice

These forms are templates only and are not legal advice. Construction laws vary significantly by state and locality. Always:

- Consult with a licensed attorney before using
- Verify current laws in your jurisdiction
- Ensure compliance with your state's specific requirements
